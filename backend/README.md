## Description

Backend was built using [Nest](https://nestjs.com/), a progressive Node.js framework for building efficient, reliable and scalable server-side applications.

## Installation

```bash
# installs dependencies
$ yarn
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e
```
